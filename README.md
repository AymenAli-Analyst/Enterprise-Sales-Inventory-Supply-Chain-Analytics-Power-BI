# 📖 Project Overview

This project is a complete Business Intelligence solution developed in **Microsoft Power BI** for **Almarai**.

The dashboard transforms raw business data into executive insights through interactive visualizations, advanced DAX calculations, and KPI-driven analytics.

It enables decision-makers to monitor financial performance, evaluate sales trends, analyze customer behavior, optimize product performance, and improve inventory management.
