# 📖 Project Overview

This project is an end-to-end Business Intelligence solution developed for **Almarai** using **Microsoft Power BI**.

The solution integrates **SQL**, **Power Query**, **DAX**, and **Star Schema Data Modeling** to transform raw business data into executive insights.

The dashboard enables business leaders to monitor financial performance, analyze sales trends, evaluate customer behavior, optimize product performance, and improve inventory and supply chain operations through interactive visualizations and KPI-driven reporting.
